Criando Scripts

Existem duas formas principais de integrar o JavaScript ao HTML e CSS:

  • Através de um arquivo externo (mais comum e recomendado), usando a tag script com parâmetro <script src="arquivo.js"></script>
  • Através da tag <script></script>, colocando o conteúdo entre as tags

Em ambos o caso, a tag deve ser aberta antes do fechamento da tag "body", depois de todo código html. No caso do arquivo externo, lembrando que ele deve estar salvo no mesmo ambiente que o index, e deve ser referenciado de forma relativa.

Formulários e Manipulação de Eventos

Para iniciarmos esse tópico, é importante definir o que é um formulário.

Formulários são a principal interação com os usuários em sistemas web (usado para fazer cadastros, buscas de dados, etc.). A tag HTML utilizada para criar um é a &lt;form>.

Seu funcionamento padrão pode ser basicamente dividido em dois itens:

  • Botão submit - envia os dados ao servidor
  • Botão reset - limpa os dados do formulário

Já no seu funcionamento atual, temos os seguintes passos:

  • O JavaScript intercepta os dados
  • Envio de dados ao servidor feito de forma assíncrona
  • Tag <button> também é muito utilizada

Seu principais campos de input (entrada de dados) são:

  • <textarea> - Usado para grandes quantidades de texto
  • <input> - elemento mais utilizado e contém:
    • Caixas de texto simples
    • Data/hora
    • Checkboxes
    • Cores
    • Números
    • E-mail
    • Botões submit e reset
    • Outros

É importante destacar que a renderização varia conforme o navegador.
