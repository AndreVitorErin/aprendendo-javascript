Alguns comandos

var s = "Teste"
var n = 1543.5

  • s.length - conta quandos caracteres tem a string, retorno = 5
  • s.toUpperCase() - passa o conteúdo para letras maiúsculas, retorno = TESTE
  • s.toLowerCase() - passa o conteúdo para letras minpusculas, retorno = teste
  • document.write() - escreve na tela do navegador
  • document.writeln() - escreve na tela do navegador e pula uma linha
  • n1.toFixed(2) - fixa a quantidade de casas decimais em 2, retorno = 1543.50
  • n1.toFixed(2)replace(".",",") - substitui um elemento por outro, nesse caso troca o ponto pela vírgula, retorno = 1543,50
  • n1.toLocaleString('pt-br', {style: 'currency', currency: 'BRL'} ) - fixa o valor no formato em reais (moeda brasileira), retorno = R$ 1.543,50
  • n1.toLocaleString('pt-br', {style: 'currency', currency: 'USD'} ) - fixa o valor no formato em dólares (moeda americana), retorno = $ 1.543,50
  • n1.toLocaleString('pt-br', {style: 'currency', currency: 'EUR'} ) - fixa o valor no formato em euro (moeda brasileira), retorno = € 1.543,50
  • window.alert - aparece um alerta e você precisa clicar em "ok" para carregar o conteúdo
  • window.confirm - aparece uma mensagem e você tem a opção de concordar ou cancelar
  • window.prompt - aparece uma mensagem e espaço para entrada de dado através do teclado
