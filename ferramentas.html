Ferramentas para Trabalhar com JavaScript

Navegador

Se você está lendo esse texto agora, muito provavelmente você já possui a acesso a internet e tem um navegador instalado no seu computador, e é importante ressaltar que para programar é importanter ter acesso a um computador, apenas celular / tablet não basta.

Como recomendação de navegador, sugerimos o Google Chrome por ter alguns recursos interessantes para criação em JS.

Visual Studio Code

Existem muitos editores que código que podem ser utilizados, mas sugerimos o Visual Studio Code (VSC ou VSCode) por ele possuir excelentes funcionalidade com relação ao JavaScript, e elas podem ser extendidas através de plugin e extensões. Além de ter uma ótima integração com o node.js e a facilidade que ele tem de criar ambientes internos.
