Ferramentas para Trabalhar com JavaScript

Navegador

Se você está lendo esse texto agora, muito provavelmente você já possui a acesso a internet e tem um navegador instalado no seu computador, e é importante ressaltar que para programar é importanter ter acesso a um computador, apenas celular / tablet não basta.

Como recomendação de navegador, sugerimos o Google Chrome por ter alguns recursos interessantes para criação em JS.

Visual Studio Code

Existem muitos editores que código que podem ser utilizados, mas sugerimos o Visual Studio Code (VSC ou VSCode) por ele possuir excelentes funcionalidade com relação ao JavaScript, e elas podem ser extendidas através de plugin e extensões. Além de ter uma ótima integração com o node.js e a facilidade que ele tem de criar ambientes internos.

Aliás, uma boa dica de extensão é o "watch in chrome" (abre os arquivos no chrome, mas através de um microservidor, o que ajuda bastante nos estudos. Para acessar essa funcionalidade, dar o comando ctrl + shft + p e digitar na área de busca que vai aparecer "watch in chrome", clicar na opção e dar ok) e o "node exec" (habilita a tecla F8 para rodar seu código JavaScript pelo node.js)

Algumas teclas de atalho interessantes no VSC:

  • Para duplicar linhas - com o cursos em cima da linha que você quer duplicar, pressionar alt + shft + seta para cima ou para baixo
  • Para excluir uma linha - ctrl + shft + k
  • Para mover uma linha - segura alt e navego com as setas
  • Renomear uma ocorrência (mudar de uma tag para outra) - com o cursos em cima da tag que você deseja renomear, clique em F2 e digite a nova tag
  • Renomear mais de uma ocorrências do mesmo estilo - ctrl + d para ir selecionando todas as ocorrências que você quer alterar (a seleção é feita em sequência)
  • Para já criar uma teg com uma classe - nomedatag.nomedaclasse + tab
    Por exemplo: div.minhaclass dará origem a <div classe="minhaclasse"><div>
